# How to Install Tindev:  

## Linux Debian Base:  
  Install Curl to add nodejs in PPA  
  ```sudo apt-get install curl```   
  To add PPA use the command:  
  ```curl -sL https://deb.nodesource.com/setup_10.x | sudo -E bash -```
